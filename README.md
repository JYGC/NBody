DESCRIPTION:
This is a simple gravity simulator using OpenGL (FreeGLUT)

DEPENDENCIES:
- FreeGLUT
- clang (gcc does not work)
- gmake

SUPPORTED OPERATING SYSTEMS:
- OpenBSD
- FreeBSD
- Fedora
- (Any UNIX based operating system should work)

COMPILING:

$ cd NBody

$ gmake

RUNNING:

$ cat (input file) | ./nbody
